1CP

REAPER’S REFLECTION

Ye who are probed to assail shall surely receive furtherance from thy brethren.

Use this Stratagem at the start of the Shooting phase. Select a friendly X’Tal unit. Until the end of the phase, you can reroll all failed hit rolls for all friendly units within 6“ of the selected unit.

1CP

IMPERMEABILITY

Though we are fragile, we shatter not in vain.

Use this Stratagem before rolling for saves. Afterwards, you may reroll all failed save rolls for that unit until the end of the phase.

1CP

CRYSTAL SABER

Thy path is thy whetstone.

Use this Stratagem at the beginning of any Fight phase. Select a friendly X’Tal unit. Until the end of the Fight phase, on a 6+ wound roll, the selected unit will inflict an additional wound.

1CP

CATALEPSEAN OVERDRIVE

Following thine ally into battle is an instinct engineered into thyself and obeyed with zeal.

Use this Stratagem before the Charge phase. Select a friendly X’Tal unit. Any friendly units within 6” of the selected unit may Ignore the roll for the Charge phase and automatically charge the maximum distance for which that unit allows.

1CP

PREOMNOR APPETITE

If an impulse for bloodshed o’ertakes thee, fear not. Embrace it.

Use this Stratagem at the beginning of any Fight phase. Select a friendly X’Tal unit. Roll a 1D6, on a 4+, that unit can add 1 to their Attack characteristics until the end of the Fight phase.

1CP

GLACIAL ADVANCE

Whilst trudging through certain areas, an X’Tal army can simply be mistaken for a glacier progression of abnormal speed.

Use this Stratagem before advancing. As a result, all units can move an additional 2“.

1CP

Ocular Obstruction

With a shimmering glare from our sourceful light, thy vision dims with darkness.

Use this Stratagem during your movement phase. Select a X’tal Glass unit. Until the start of your next Movement phase, enemy models can only shoot this unit if it is the closest visible target.

1CP

Glass Ribbons

Retaliation is but an option for even when defeat looks inevitable.

Use this Stratagem in your Movement phase. Select a Silexia unit from your army that is within 1” of an enemy unit. That unit can Fall Back, shoot and charge in this turn.

1CP

Merciless

Until victory is assured, ye shall remain relentless!

Use this Stratagem when a Perlumite unit is chosen to attack in the Fight phase. Each time you make a hit roll of 6+ for a model in this unit during this phase, it can immediately make an extra attack against the same target using the same weapon. These extra attacks cannot themselves generate any further attacks.

1CP

Test Thy Will

Only through thy failure can ye become successful.

You can use this Stratagem when a friendly X’tal unit is chosen to fight. If you do so, you can re-roll any failed wound rolls for the unit if the target of the attack has the GEM keyword.
